HTML Structure General layout and content structure

Origami is a multi purpose theme that is great for anything from creative portfolios to corporate websites. There are 4 main sections I want to summarize here. There is the navigation, the services section, the meet the team section and the pricing section.

Navigation

The navigation is a sidebar navigation made up of a Logo and Menu items. The menu items also have spaces for icons and can also be set up for smooth scroll or gallery filtering.

To change links in the menu, change the li tags in ul tag with class nav.

Elements HTML Elements

Bolt supports many cool customized bootstrap elements. I will go over elements including the Bootstrap Column system, forms, Collapsible toggle menus, alert boxes, buttons and Modal image previews.

Bootstrap Columns

Bootstrap columns are the easiest way to make things responsive. just define a row div, and you can add columns to them. with tags like col-sm-4, that means you can fit 3 of those into a column on a small device. There are 12 columns in a row. Usually you want to define col-xs-12 as 12 columns because on small screens things will shrink.

  <div class="row">
  <div class="col-md-4 col-sm-4 col-xs-12">
</div>

Alerts

Alerts are banners that should convey a message and also be dismissible.

There are 5 different kinds including the normal alert. There is a alert-warning, alert-error, alert-info, and alert-success

Buttons

Buttons are pretty simple to use, just define an anchor with the class "btn". There is also a larger size if you add the class "btn-large" to the btn.

You can also add icons into the button.

JavaScript Files

Tag Description

MixItUp Portfolio / Gallery

The gallery uses a jQuery plugin called mixitup where you can tag items with certain categories which you can then hide and show depending on specific categories.

You can assign categories to your gallery items by adding a category name as a class to the li tags with class mix

Masonry Blog

Masonry is a plugin that allows things in a row to fit nicely together without unnecessary whitespace.

Here is the initializing code. If you wanna customize any masonry options it would go inside the $container.masonry({ block.

  var $container = $('.masonry-grid');
  
  $container.imagesLoaded(function(){
    $container.masonry({
      itemSelector : '.item',
      gutter : 0
    });
    $(".item").css("opacity", "1");
  });
